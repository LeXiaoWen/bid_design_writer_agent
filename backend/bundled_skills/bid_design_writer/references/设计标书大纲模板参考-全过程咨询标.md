## 使用说明

本模板适用于「全过程工程咨询服务」类型的投标项目，设计部分是其中的一个板块（非独立的设计标）。使用时按以下原则处理：

1. **图片策略由招标文件决定**：每张图的处理方式（占位 / 自生成提示词）唯一依据是阶段一从招标文件中提取的图纸分类，不以本模板中的章节名推断。本模板中标注的「输出方式」为参考，实际以招标文件图纸要求为准。
2. **文字填充**：基于阶段一提取的项目信息、设计理念、策略等填充各章节内容
3. **章节调整**：如招标文件评分项要求不同，以评分项为准调整章节划分
4. **评分标注**：每章开头标注 `> 📌 本章响应评分项：...`（与 SKILL.md 规则一致）
5. **区别于纯设计标**：此类标书设计部分篇幅有限（约 25-35 页），侧重管理措施而非效果图展示，少用图片多用文字论述

## 第三章 组织机构、管理机制及相关资源配置

> **内容类型**：文字为主 | 全部编写完整

### 3.1 组织机构体系
- 项目总负责人统筹管理
- **矩阵式管理**：公司各职能部门每周例会，对已完成工作总结，对下周工作依据实际进度调整
- 领导小组按需召开专项协调会议，动态调配资源

### 3.2 项目管理机制
- 内部质量管理体系（ISO 体系）：质量方针→质量目标→职责权限→资源控制→内部沟通→产品实现→过程监视测量→数据分析→管理评审→纠正预防措施→持续改进

### 3.3 项目人员配备要求
- 各专业技术骨干任各专业负责人及设计工程师
- 设计人员保持相对稳定，未经业主同意不随意更换
- 设计方案、选材上力求优化、严格控制投资
- 结构专业：通过方案调整优化用钢量、混凝土用量

### 3.4 设计工作资源配置
- 每位设计人员配备优良品牌电脑，双屏配置
- 计算机出图、打印、复印、晒图等设施设备齐全

### 5.1 全过程造价控制措施
- 重视工程施工前的图纸会审，避免施工中发现图纸问题而增加变更
- 分清签证权限，加强施工变更签证的审计管理

### 5.2 各设计阶段造价控制（按专业展开）

| 专业 | 控制策略 |
|------|----------|
| **建筑专业** | 设计各阶段合理控制建设规模；合理规划建筑功能布局、层高和平面系数；**全寿命周期角度控制工程造价**，考虑建成后运营和维护费用；优化管线布置节约设备管线成本；提前进行节能计算，选择保温材料 |
| **结构专业** | 结构体系优化，构造合理；"安全、经济、合理"设计原则；**多方案试算分析比较**择优选择结构布置；**多基础方案技术经济比较**（含地基处理、基坑支护、降水等措施费）；精确计算、精细设计、精心选择材料和构造措施 |
| **给排水专业** | 各系统合理合规且成本经济；合理运用成熟新技术、新材料为客户节约造价；**不同使用要求的部门采用不同设计参数和设备选型**；尽可能重力自流减少抽升排放 |
| **电气专业** | 高低压变配电系统及干线系统合理经济；变压器/发电机容量选择合理经济；应急电源选型合理经济；高低压变配电室及电气竖井设置合理经济；**优化电缆敷设方式减少电缆降容**，降低电缆使用量；建立电气设备配置表供业主选择配置标准 |
| **暖通专业** | 系统选型节能经济；设备参数匹配实际负荷避免过度设计 |
