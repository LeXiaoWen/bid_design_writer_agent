# 可复用模块卡片

本文件用于把真实投标材料中的有效表达方式沉淀为跨项目可调用的模块。模块不是固定目录，也不是可直接粘贴的文案；每次调用前必须核对适用条件、招标评分、A 级约束和项目事实。

## 卡片格式

```markdown
### {模块名称}
- 适用条件：{项目类型/评分项/资料前提}
- 不适用条件：{何时不得使用}
- 解决的评分问题：{评分项与满分证据}
- 最低交付物：{文字、图纸、效果图、表格、节点}
- 关键控制点：{控制对象—触发节点—输出记录—闭环}
- 不可迁移内容：{项目指标、产品、材料、企业信息、承诺}
- 来源：{真实材料文件与页码}
```

## 住宅方案深化模块

### 滨水/景观资源 + 场地高差的住宅方案深化

- 适用条件：住宅项目；已有概念方案；评分关注项目理解、场地分析、总平、产品或鸟瞰；场地存在可利用的景观资源或竖向高差。
- 不适用条件：平坦场地、非居住项目，或评分不要求方案深化表现。
- 解决的评分问题：项目理解、场地分析图、彩色总平、总体鸟瞰。
- 最低交付物：资源/约束分析图、竖向与视线示意、方案生成逻辑、彩色总平表现版、工程校核版总图、总体鸟瞰提示词。
- 关键控制点：以正式地形和规划条件为输入；将景观视廊、土方、排水、消防和道路高程在总图中叠图校核；形成“竖向/视线/流线问题清单”并关闭。
- 不可迁移内容：具体江景、城市名称、标高、高差、建筑密度、面积和主题名称。
- 来源：宜宾项目真实方案中“项目背景、场地分析、规划设计”页组；仅沉淀方法。

### 住宅地下室六联动优化

- 适用条件：评分含地下室布局、指标优化或施工图可实施性；项目有地下车库。
- 不适用条件：无地下室项目。
- 解决的评分问题：已有方案优化、施工图重点难点、成本控制。
- 最低交付物：柱网/车位/坡道/机房/管综/归家流线六联动分析图，净高控制表，优化前后对比表。
- 关键控制点：以车位指标、消防、结构柱网、设备条件和正式竖向为输入；在条件图阶段组织建筑、结构、机电叠图；输出碰撞清单、净高问题清单和关闭记录。
- 不可迁移内容：具体车位数、会所面积、坡道位置、地下室边界及任何未经确认的效率指标。
- 来源：宜宾项目真实方案“地下室专篇”及施工图大纲的专业对图方法；仅沉淀方法。

### 住宅立面落地与成本协同

- 适用条件：评分要求外立面落地性、材料/成本分析或建筑形象；已有方案或产品分级策略。
- 不适用条件：招标范围不含立面深化。
- 解决的评分问题：已有方案优化、成本控制、方案完整性。
- 最低交付物：立面策略说明、材料分区表、标准层节点占位、产品差异化矩阵、成本替代逻辑、立面效果提示词。
- 关键控制点：材料、分格、节点、设备遮蔽、排水、保温和维护同步审查；形成样板/材料/节点/成本四表；未获确认的材料和价格不得写为定案。
- 不可迁移内容：具体材料品牌、色号、幕墙/窗墙选型、户型编号、单价和成本承诺。
- 来源：宜宾项目真实方案“外立面落地性及成本分析”页组；仅沉淀表达结构。

### 户型优化评分表达

- 适用条件：评分明确要求户型优化、产品创新或居住体验；已取得当前户型资料。
- 不适用条件：未取得户型资料、只做施工图无产品优化权限。
- 解决的评分问题：户型优化、项目理解、方案深化。
- 最低交付物：户型配比表、典型户型平面占位、优化前后对比、景观/采光/通风/收纳/设备/动线六维说明。
- 关键控制点：户型调整须同时复核面积、规范、结构、机电、报批和成本影响；输出“户型调整影响表”。
- 不可迁移内容：具体户型面积、房间数、得房率、产品命名和卖点文案。
- 来源：宜宾项目真实方案“产品篇”页组；仅沉淀六维表达法。

## 施工图设计大纲模块

### 施工图质量闭环

- 适用条件：评分包含设计质量保障；多专业施工图项目。
- 解决的评分问题：设计质量保障、重点难点、后续服务。
- 最低交付物：设计输入清单、条件图确认单、专业接口问题清单、校审记录、审图意见闭环表。
- 关键控制点：设计输入核对 → 条件图冻结 → 专业对图 → 校审 → 审图回复 → 归档；每步须有责任角色、关闭日期和复核人。
- 推荐节点：在施工图完成约 75% 时组织专业对图；具体比例可按项目工期调整。
- 不可迁移内容：企业制度名称、固定组织架构、人员姓名和认证表述。
- 来源：宜宾项目真实施工图设计大纲的专业对图、校审与质量管理内容；仅沉淀流程。

### 施工图进度、成本与服务联动

- 适用条件：评分包含进度、成本和后续服务；设计服务工期明确。
- 解决的评分问题：进度保障、成本控制、服务承诺。
- 最低交付物：倒排进度表、风险台账、成本控制台账、周报模板、变更/现场服务闭环表。
- 关键控制点：里程碑与成果挂钩；重大风险当日升级；每版关键成果同步校核成本影响；变更须记录原因、专业影响、工期影响和关闭结论。
- 不可迁移内容：具体天数、响应时限、驻场频次、企业资源规模；仅在投标人确认履约能力后写入。
- 来源：宜宾项目真实施工图设计大纲的进度、成本、服务章节；仅沉淀流程。

## 教育建筑设计标模块

### 职业教育校园的山地/滨水场地深化

- 适用条件：职教、高校或培训校园；场地存在高差、边坡、河道/绿地/既有校区衔接；评分含场地分析、总体布局、竖向、交通消防或设计理念。
- 不适用条件：无校园功能、无场地条件资料，或招标不含方案深化。
- 解决的评分问题：项目理解、总体布局与竖向、建筑造型、方案完整性。
- 最低交付物：区位与周边资源分析、现状/边坡/水系约束分析、竖向与土方逻辑、功能组团图、人车后勤消防流线图、主/次景观轴剖面、日照结论及依据。
- 关键控制点：以规划条件、地形和地勘为输入；将边坡、河道/防洪、道路高程、消防、排水和既有校园接口列入同一风险清单；日照、消防、竖向结论均应有软件/规范/图纸依据。
- 不可迁移内容：具体学校名称、河道名称、交通节点、容积率、用地面积、建筑数量、高差、日照时数及周边配套描述。
- 来源：重庆青年职业技术学院真实技术标的前期分析、方案生成、总体规划页组；仅沉淀图文证据结构。

### 既有校园风貌延续与低碳立面

- 适用条件：扩建/改扩建校园；任务书要求与既有校区协调，或评分考察造型、绿色低碳、立面设计。
- 不适用条件：无既有校区资料或任务书未要求风貌延续。
- 解决的评分问题：建筑造型、低碳/绿色建筑、方案完整性。
- 最低交付物：既有建筑语汇提取图、材料/色彩/尺度关系表、节能立面策略图、典型节点占位、单体效果图或提示词。
- 关键控制点：从既有校区资料中提取可验证的材料、尺度、色彩和构图语汇；遮阳、通风、围护、屋顶/雨水等低碳措施均须与气候条件、规范和机电条件核对。
- 不可迁移内容：特定砖色、幕墙/涂料品牌、特定角度遮阳构件、具体光伏/绿化比例及未经计算的节能指标。
- 来源：重庆青年职业技术学院真实技术标的立面设计页组；仅沉淀“风貌提取—现代转译—性能校核”的表达结构。

### 效果图硬约束与 A3 技术标页组

- 适用条件：招标文件明确效果图数量/视点，或要求提交方案文本、效果图文本；技术标以横向 A3/PDF 页面评审为主。
- 不适用条件：仅递交纯文字技术方案，或未要求视觉成果。
- 解决的评分问题：效果图、方案完整性、场地/总平/造型等主观评分项。
- 最低交付物：效果图合规清单、评分—页组—视觉资产矩阵、视觉锚点表、章节扉页、每个主观评分项的单页主证据脚本、图注与结论文字。
- 关键控制点：一项硬性视点对应一条提示词/一张成图；提示词按“视点—前中后景—建筑锚点—场景光线—表达方式—禁止项”组织，并与总图、单体平面、立面材料一致；标题、项目名、页码和定位小图从渲染提示词中剥离，页组按“章节扉页—主图—一句结论/关键数据”组织，避免一页堆叠多个主结论。
- 不可迁移内容：具体画幅、页数、配色、主题、渲染图、建筑形象和项目中英文字样；以本项目招标格式为准。
- 来源：重庆青年职业技术学院真实技术标的 A3 横向版式、建筑表现与总体规划页组；仅沉淀版式证据方法。
