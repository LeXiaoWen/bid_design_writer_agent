# 设计标书大纲模板参考（12章通用框架）

> **模板来源**：重庆青年职业技术学院改扩建项目 W27-3/03 地块设计方案（中标技术标书）  
> **适用场景**：当招标文件**未提供**明确的设计方案模板/目录结构时，作为阶段二的章节组织参考  
> **适用类型**：建筑工程设计类标书（学校/医院/办公楼/产业园等）  
> **使用方式**：图片用占位符替代，只补充文字内容；章节名称和顺序可根据实际项目微调

---

## 使用说明

1. **图片策略由招标文件决定**：每张图的处理方式（占位 / 自生成提示词）唯一依据是阶段一从招标文件中提取的图纸分类，不以本模板中的章节名推断。本模板中标注的「输出方式」为参考，实际以招标文件图纸要求为准。
2. **文字填充**：基于阶段一提取的项目信息、设计理念、策略等填充各章节文字内容
3. **章节调整**：如招标文件评分项要求不同，以评分项为准调整章节划分
4. **评分标注**：每章开头标注 `> 📌 本章响应评分项：...`（与 SKILL.md 规则一致）
5. **项目适配**：模板中的示例文字替换为实际项目信息，不照搬

## 2. 前期分析（Preliminary Analysis）

> **内容类型**：图文混合 | 图片策略由招标文件图纸要求决定

- **区位分析**：地理位置、交通条件（公交/轨道/快速路）、周边配套（商圈/医院/自然景观）、区域定位
- **上位规划**：用地性质、规划指标（容积率/密度）、周边用地情况分析
- **现状分析**：场地现状描述（地形/建筑/植被/水系）、周边条件、交通条件、重点注意事项
- **现状地形分析**：地貌类型、高差、坡度、地质条件、场地适宜性评价
- **重难点分析**（前期阶段）：从场地条件、功能需求、约束条件等维度识别 4-5 个核心难点

---

## 3. 设计理念（Design Concept）

> **内容类型**：图文混合 | 图片策略由招标文件图纸要求决定

- **设计思考**：基于项目定位提出核心问题 → 多维度分析框架
- **设计目标**：围绕核心问题提出 4-6 个具体目标
- **设计理念**：核心理念的关键词/图形化表达 + 理念阐述
- **设计愿景**：一句总结 + 关键词标签（如"共享·绿色·创新·智慧"）
- **特色模式**（按项目类型）：如产教融合模式、医养结合模式、生态园区模式等

## 4. 方案生成（Scheme Generation）

> **内容类型**：图文混合 | 图片策略由招标文件图纸要求决定

| 步骤 | 核心内容 | 输出方式 |
|------|----------|----------|
| 入口选择 | 分析各方向城市道路条件，确定主入口/次入口/后勤入口位置及理由 | 文字 + 由招标文件图纸分类决定 |
| 动静分区 | 根据外部条件划分动区（临路展示面）和静区（临景观面） | 文字 + 由招标文件图纸分类决定 |
| 空间轴线 | 确定主要轴线布局（如文化轴+生态轴的双轴格局） | 文字 + 由招标文件图纸分类决定 |
| 体块推演 | 4-6步推导：基础体量→体量拆分→体块错落→镂空/架空→连廊串联 | 文字 + 由招标文件图纸分类决定 |
| 引景入园 | 利用外部景观资源的策略（观景平台/低矮建筑/视线通廊） | 文字 + 由招标文件图纸分类决定 |
| 形象界面 | 各方向立面的差异化处理策略与材质组合 | 文字 + 由招标文件图纸分类决定 |

## 5. 总体规划（Master Plan）

> **内容类型**：图文混合 | 图片策略由招标文件图纸要求决定

- **总平面图**：由招标文件图纸分类决定（含经济技术指标表）
- **功能体块**：功能分区（列出各功能区及建筑单体面积/层数）
- **流线分析**：人车分流策略、主次入口、校内流线组织
- **消防分析**：环形消防车道（宽≥4m/转弯半径≥12m）、消防出入口、应急避难区
- **日照分析**：分析软件/标准/结论（关键功能用房日照达标情况）
- **绿色校园/园区**：屋顶绿化 + 垂直绿化 + 庭院景观
- **共享校园/园区**：共享空间策略（开放界面/共享中庭/休闲平台/生态景观/形象入口）
- **轴线剖面**：由招标文件图纸分类决定 主轴线剖面 + 次轴线剖面
- **模块设计**：灵活空间的标准化/可变模式
- **智慧校园/园区**：智慧安防（防线层级）+ 智慧教学/办公（智能化设施）

## 7. 立面设计（Elevation Design）

> **内容类型**：图文混合 | 图片策略由招标文件图纸要求决定

- **低碳节能立面**：6项策略（光伏发电/高性能围护结构/自然通风/绿色屋顶/雨水收集/遮阳系统），原则：被动优先·高效利用·低碳运营·生态融合
- **新旧/地域文化融合**：与原有建筑风格的延续关系（材质提取/色彩/建筑语汇）
- **立面效果展示**：由招标文件图纸分类决定 + 风格文字描述
- **立面图（东南西北）**：由招标文件图纸分类决定 ×4
- **核心风格特征说明**：材质与色调 / 经典语汇现代转译 / 体块组合虚实对比

## 8. 设计管理（Design Management）

> **内容类型**：文字为主 | 6大保障措施，全部编写完整文字

| 保障措施 | 核心要点 |
|----------|----------|
| 质量保障 | 三级审核机制（自审→专业审核→终审）、专属设计标准、专业协同会议、质量复盘、人员培训与绩效考核 |
| 成本保障 | 成本管控小组、方案经济性比选、概算精准管控、成本动态跟踪（周报）、超支5%预警机制 |
| 进度保障 | 细化到日/周的进度计划、两级审核（24h+48h）、交叉作业、滞后24h预警/48h提级处理 |
| 保密保障 | 专项保密制度、全员保密承诺书、涉密资料全流程闭环（纸质+电子）、评审封闭管理、技术防护与应急处置、终身保密义务 |
| 安全保障 | 设计安全管控体系、方案安全把控（红线/退让/容积率）、初步设计技术安全、三级安全审核、过程管控与风险应急 |
| 其他保障 | 组织保障（上下联动体系）、人员保障（固定团队+备用人员）、技术保障（专家论证+规范更新）、资源保障（软硬件+专项经费） |

## 10. 危大工程专项设计（Hazardous Engineering）

> **内容类型**：文字为主 | 9个子章节，全部编写完整

| 章节 | 核心内容 |
|------|----------|
| 第一章 总则 | 编制目的、原则、依据（住建部令第37号/建办质〔2018〕31号/地方细则） |
| 第二章 工程总体概况 | 项目概况（功能/层数/高度/结构形式）、危大工程识别与界定（基坑边坡/大跨度支撑/高处临边/特殊环境）、场地风险特征总结 |
| 第三章 设计控制条件与荷载取值 | 三大控制工况划分、荷载取值标准（永久/可变/特殊荷载）、安全控制限值 |
| 第四章 各专项危大工程设计方案 | 基坑边坡（支护+截排水+防冲刷三位一体）、大跨度梁板（截面优化+支撑体系参数+施工控制）、临边防护（刚性防护+防坠物）、雨季专项 |
| 第五章 结构专项计算与验算说明 | 边坡基坑验算、大跨度结构验算（使用+施工两阶段）、临边防护验算 |
| 第六章 安全监测专项设计 | 监测项目（位移/沉降/裂缝/水位/挠度/应力）、监测频次与三级预警值、成果要求 |
| 第七章 施工阶段设计管控硬性要求 | 5+N条强制性管控条款（严禁擅自变更/先排水后开挖/强度达标拆模/临街封闭防护/专家论证） |
| 第八章 风险辨识与设计防控措施 | 四类核心风险辨识 + 六大设计防控措施 |
| 第九章 验收与资料管控设计要求 | 分阶段验收标准（基坑支护/支架体系/大跨度隐蔽/临边防护/雨季专项） |

## 11. 重难点分析（Key Points and Difficulties Analysis）

> **内容类型**：文字为主 | 按专业分节，每专业=重点难点 + 应对措施

| 专业 | 常见重点难点方向 | 应对措施方向 |
|------|------------------|-------------|
| 建筑 | 全过程协调、功能复杂性、防水节点、遮阳与立面结合、规范更新 | 专业早期介入、专项设计组、协同机制、BIM技术、新规培训 |
| 结构 | 抗震设防等级（乙类）、大跨度结构、连体/连桥设计 | 提高一度设防、方案经济比选（预应力/钢骨/钢梁）、结构缝分开 |
| 电气 | 变配电系统、智能化整体架构、变压器容量 | 咨询供电部门、深入负荷计算、开放性可扩展系统 |
| 暖通 | 防排烟自然排烟、节能、设备房散热、气味控制 | 自然通风优先、低噪高效设备、CO传感器联动、活性炭吸附 |
| 给排水 | 消防系统设计、给水分区、管线综合协调、设备噪声 | 充分了解规范、机电管线综合/碰撞检测、合理选材 |

## 12. 设计说明（Design Description）

> **内容类型**：文字为主 | 10个子项，每个均需包含设计依据规范清单

| 子项 | 核心内容 |
|------|----------|
| 方案设计说明 | 设计依据（15-25项规范列表）、工程概况、规划设计条件与目标、总平面设计（原则+构思+特色+交通）、建筑设计（理念+平面+造型立面+主要经济指标） |
| 结构设计 | 规范依据、活荷载取值表、结构选型（框架/基础/抗震参数/场地条件）、材料选用、新技术 |
| 给排水设计 | 设计范围、给水系统（水源/水量/方式/计量）、排水系统（体制/管材）、雨水系统（暴雨强度/重现期）、消防系统（水量/消火栓/喷淋/气体/灭火器） |
| 电气设计 | 变配电系统（负荷分级/电源/容量/变配电所）、照明系统、防雷接地、弱电智能化各子系统 |
| 暖通空调设计 | 空调方案（各区域形式）、通风设计（换气次数表）、消防防排烟（自然/机械/控制）、节能 |
| 消防设计 | 总体消防（类别/耐火等级/消防车道/防火间距）、建筑单体消防（疏散/防火分区/消防控制室） |
| 智能化设计 | 安全防范/信息管理/远程数据采集/电子监控/停车场管理/公共设施管理/智能照明 |
| 节能设计 | 规划节能（朝向/间距/通风/绿化）、单体节能（体形/窗墙比/遮阳）、围护结构、节水（器具/计量/减压）、暖通节能、电气节能（变压器/照明/电梯） |
| 绿色建筑设计 | 绿建评价标准依据、规划设计（地形/光污染/风环境/噪声）、各专业绿建技术措施 |
| 海绵城市设计 | 下沉式绿地（下沉100-200mm/溢流口）、雨水花园、透水铺装、绿化屋面、雨水收集（弃流→沉淀→过滤→消毒→回用） |
